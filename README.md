 # Diabetes Prediction API (FastAPI + Random Forest)

This project exposes a machine learning model (Random Forest) for diabetes prediction using FastAPI.

## 🚀 Setup

Clone the repository:
   ```bash
   git clone https://github.com/Suprith07112047/mlops-practice.git
   ```
Navigate to the project directory:
   ```bash
   cd mlops-practice
   ```
Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
